# Ashwika Bhatnagar — Portfolio

Personal portfolio website built with Next.js 14, TypeScript, and Tailwind CSS.

🔗 **Live:** [portfolio-eosin-seven-65.vercel.app](https://portfolio-eosin-seven-65.vercel.app)

## Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

## Structure

All content lives in a single file — `lib/data.ts`. Edit that file to update the entire site.

## Sections

- Hero · About · Skills · Projects
- Achievements · Experience · Education
- Certifications · Contact

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploying

```bash
npx vercel --prod
```
